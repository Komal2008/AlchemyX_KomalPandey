# NewsQuest

NewsQuest is a gamified news and learning platform with a Vite + React frontend in `frontend/` and a separate `backend/` service.

## Structure

- `frontend/` - main frontend application and static assets
- `backend/` - API and service code

## Scripts

- `cd frontend && npm run dev`
- `cd frontend && npm run build`
- `cd frontend && npm run lint`
- `cd frontend && npm run test`
